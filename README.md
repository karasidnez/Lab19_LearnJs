## Описание
В ходе выполнения лабораторной работы было проведено знакомство с языком программирования JavaScript, его местом в веб-разработке и сравнение базовых концепций с C#.  

### Что изучили:
- Циклы: `for`, `while`, `do...while`, операторы `break` и `continue`, вложенные циклы.
- Функции: классическое объявление, параметры по умолчанию, функции как значения, стрелочные функции.
- Массивы: создание, методы `push`, `pop`,`indexOf`, `includes`, цикл `for...of`.
- Объекты: созднае, доступ к свойствам, добавление/удаление свойств, методы объектов, перебор свойств (`for...in`), вложенные объекты и массивы.
- Практическое применение полученных знаний через выполнение заданий и фиксация результатов в системе контроля версий.
